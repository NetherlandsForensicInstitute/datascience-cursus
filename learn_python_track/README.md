## The "Learn Python" Track
This track is divided into two distinct sections:
1.  **Weeks 1–3:** Interactive learning via [futurecoder.io](https://futurecoder.io).
2.  **Weeks 4–5:** Data analysis notebooks using the **pandas** package.

### General Notes on futurecoder.io
* **Completion:** You are expected to finish the weekly content before the next tutorial. The weekly content you are expected to do can be found in the README of each week. Expect to do some work at home to supplement in-person sessions.
* **Accounts:** Sign up with your e-mail address (via the "Start learning" button) to save your progress.
* **Progress:** Use the "Table of Contents" to track your standing.
* **Challenges:** Bonus/optional exercises are not mandatory. You can skip them using the links provided in the text.
* **Getting Help:**
    * Check **Requirements** and **Assessment** sections for live explanations.
    * Use the **Hints and Solution** section. Use hints liberally to get moving, but view full solutions sparingly. 
    > **Note:** Programming is learned by doing. Ensure you understand a solution before moving on.
